# InvenTree Demo Environment

## Demo URL

**https://demo.inventree.org**

## User Accounts

| Username | Password | Access Level |
|----------|----------|--------------|
| allaccess | nolimits | Full create/edit/view permissions |
| reader | readonly | View-only access |
| engineer | partsonly | Parts management and stock viewing |
| admin | inventree | Superuser with admin functions |

## Environment Notes

- The demo database resets daily to maintain a known state.
- The system updates to the latest master branch version during maintenance periods, which may cause brief downtime.
