# Project Overview

**API Test Automation Framework (TAF)** for [InvenTree](https://demo.inventree.org/) — an open-source inventory
management system. The framework automates REST API testing for InvenTree's core domains (Parts, Categories, Stock,
Pricing, Companies) using Java, Maven, REST Assured, and TestNG.

- InvenTree API docs: https://docs.inventree.org/en/stable/api/
- InvenTree demo: https://demo.inventree.org
